## FreshGPU

Limpiar las carpetas de caché de controladores gráficos de diferentes rutas dependiendo del fabricante y muestra las rutas a donde el programa apunta.

## Running your application:

```sh
neu run

```

## Building your application:

```sh
neu build --release

```
